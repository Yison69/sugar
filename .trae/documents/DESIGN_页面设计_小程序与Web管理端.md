# 页面设计文档（桌面端优先：Web；小程序适配移动端）

## 全局设计（共通）
- Layout：Web 采用 CSS Grid（左侧固定导航 + 右侧内容区）；内容区内部使用 Flex + Card 列表。小程序以纵向堆叠为主，列表为双列卡片栅格。
- Global Styles / Tokens：
  - 颜色：背景 #F7F8FA；主色 #111827（近黑）；强调色 #2563EB（蓝）；成功 #16A34A；警告 #F59E0B；危险 #DC2626。
  - 字体：Web 默认 14/16/20/28 级；标题加粗 600；正文 400。
  - 卡片：圆角 12；阴影轻（elevation 1）。
  - 按钮：主按钮实心主色；次按钮描边；hover/press 深一阶；禁用 40% 透明。
  - 链接：主色下划线 hover。
- 交互通用：所有提交类操作（预约/保存）展示 loading；失败 toast；成功 toast + 回跳/刷新。

## 小程序：首页/分类浏览页
### Meta Information
- title：摄影服务
- description：浏览作品与服务套餐，支持预约与点赞

### Page Structure
- 顶部：品牌区（摄影师昵称/Logo 可选）+ 搜索框（可选，默认仅本地筛选）
- 分类栏：横向 Tabs（毕业照/写真照/婚纱照/场地租赁）+ 当前选中态
- 筛选条：类型切换（作品/套餐/全部）+ 排序（最新/最热）
- 内容区：卡片网格列表（小程序建议双列）
- 底部（可选）：固定入口“我的预约”

### Sections & Components
1. 分类 Tabs
   - 点击切换时刷新列表；保持滚动到顶部。
2. 卡片（作品/套餐通用）
   - 元素：封面图、标题、价格（套餐显示“起价/基础价”）、点赞按钮+数值。
   - 操作：点击卡片进入详情；点击点赞仅影响该卡片（乐观更新）。
3. 分页加载
   - 下拉刷新、触底加载下一页；空态提供“暂无内容”。

## 小程序：作品/套餐详情页
### Meta Information
- title：作品详情 / 套餐详情（随内容动态）
- description：查看详情、选择套餐选项、提交预约

### Page Structure
- 顶部：轮播图（作品为图集；套餐可用封面+示意图）
- 基础信息区：标题、分类、点赞区、基础价格（套餐）
- 内容说明区：描述/交付物/注意事项
- 套餐选项区（仅套餐）：按“选项组”卡片分段
- 底部固定操作栏：总价（含明细入口）+ 主按钮“立即预约”

### Sections & Components
1. 点赞组件
   - 状态：未赞/已赞；点击切换；需防重复提交。
2. 套餐选项组（核心）
   - 组头：名称、必选标识、选择规则（单选/多选）。
   - 选项行：名称、价格变化标签（+¥ / -¥ / 替换）、数量步进器（当允许数量）。
   - 校验：必选未选时禁用“立即预约”或在点击时提示。
3. 价格明细
   - 展示：基础价、每个选项 delta、合计；用于用户确认。
4. 预约弹窗/预约页内表单
   - 字段：姓名、手机号、预约日期时间、地点、备注。
   - 提交：成功后跳转“我的预约”并定位到新纪录。

## Web 管理端：控制台（作品/套餐/预约）
### Meta Information
- title：摄影管理控制台
- description：管理作品、套餐与预约

### Page Structure
- 左侧 Sidebar：导航（作品、套餐、预约、退出）
- 顶部 Header：当前模块标题 + 主操作按钮（新增作品/新增套餐）
- 主内容区：表格/卡片列表 + 右侧详情抽屉（桌面端）

### Sections & Components
1. 作品管理
   - 列表：封面、标题、分类、上架状态、点赞数、创建时间；支持搜索（标题）与分类筛选。
   - 编辑抽屉：基础字段、图集上传（云存储）、上架开关、保存。
2. 套餐与选项管理
   - 套餐列表：基础价、分类、上架状态。
   - 套餐编辑：
     - 基础信息：标题、分类、基础价、描述、交付物。
     - 选项组编辑器：新增/删除组；组内新增/删除选项；设置 replace/add/minus、单选/多选、必选、deltaPrice、数量上限。
     - 右侧预览：模拟用户选择后价格变化。
3. 预约管理
   - 列表：预约人、手机号、预约时间、关联套餐/作品、状态、创建时间。
   - 详情抽屉：展示用户提交快照与价格明细；状态操作按钮（确认/完成/取消）+ 管理员备注输入。

响应式规则：
- Web 在 >=1024px 采用 Sidebar + 内容双栏；<1024px Sidebar 折叠为抽屉。
- 小程序以移动端为主；图片轮播高度随屏宽自适应（16:9）。
