# 页面设计文档：管理端资源上传弹窗与小程序资源展示

## 全局设计（桌面优先）
- Layout：管理端采用 Flexbox + 局部 CSS Grid；弹窗为“固定头部 + 可滚动内容区 + 固定底部操作区”。
- Meta Information（管理端）：
  - title：套餐管理 - 资源配置
  - description：为套餐/选项上传封面与媒体资源并发布
  - og:title / og:description：同上
- Global Styles（建议 Token）：
  - 背景：#F7F8FA；卡片：#FFFFFF；边框：#E5E6EB
  - 主色：#1677FF；危险：#F53F3F；成功：#00B42A
  - 字体：14px 基础；标题 16/18/20；行高 1.4
  - 按钮：Primary/Default/Danger；hover 提升亮度 6%；disabled 降低透明度
  - 链接：主色 + underline on hover
  - 过渡：150ms ease（hover、弹窗进入/退出）

## 页面 1：管理端登录页
### Page Structure
- 居中单列卡片（max-width 420px），背景浅灰。

### Sections & Components
1. 顶部品牌区：Logo + 系统名称。
2. 登录表单：账号、密码、登录按钮。
3. 状态提示：错误信息（红色），登录中 loading。

## 页面 2：管理端套餐管理页（含编辑弹窗）
### Page Structure
- 顶部导航（可选）+ 内容区两段：列表区 + 弹窗（按需浮层）。
- 列表区为表格/卡片列表（按现有实现）。

### 核心组件：套餐编辑弹窗（关键交互）
#### Layout
- 弹窗容器：宽 840–960px（桌面），高度为视口 80vh。
- 结构：
  - Header（固定）：标题“编辑套餐”、右上角关闭 X。
  - Body（可滚动）：表单与上传区。
  - Footer（固定）：取消 / 保存（主按钮）。
- 滚动规则：仅 Body 滚动，禁止整页滚动穿透（modal open 时 body overflow hidden）。

#### 易退出（优先级高）
- 退出方式：
  1) 右上角 X
  2) Footer 取消
  3) Esc
  4) 点击遮罩（默认开启；若存在未保存变更则触发二次确认）
- 未保存变更提示：
  - 文案："你有未保存的修改，确定要退出吗？"
  - 操作：继续退出 / 返回编辑

#### Sections & Components（弹窗 Body）
1. 基础信息区（按现有字段）
   - 表单两列栅格（Grid）：左列文字字段，右列状态/开关等。
2. 封面上传区（单资源）
   - Uploader：拖拽/点击上传
   - 预览：图片缩略图（16:9 或 1:1，按业务选择）；支持“替换/删除”
   - 校验提示：格式/大小不符合时，提示原因并阻止提交
3. 套餐媒体上传区（多资源）
   - 列表展示：缩略图/文件名 + 类型标签（image/video/file）
   - 交互：上传进度条；失败重试；删除
   - 排序：拖拽排序（桌面）或上下移动按钮（兼容）
   - 预览：
     - 图片：点击打开预览层
     - 视频：内嵌播放器或新层播放
     - 文件：显示图标+下载/打开
4. 选项项资源上传区（按选项分组）
   - 结构：折叠面板 Accordion（每个选项一组）
   - 每组选项内：资源列表 + 上传按钮 + 排序/删除/预览
   - 空态："该选项暂无资源"

## 页面 3：小程序端套餐展示页（列表/详情按现有入口）
### Layout
- 小程序端以纵向堆叠为主：封面/媒体 → 选项切换 → 资源区域。
- 适配策略：
  - 图片：使用 mode="widthFix"（或按现有）保证不变形；为长图提供最大高度与“展开查看”
  - 视频：使用原生 video 组件，封面图优先使用套餐 cover
  - 文件：显示文件卡片（图标+名称+大小），提供“打开/下载”

### Meta Information（小程序）
- 页面标题：套餐详情
- 分享卡片：优先用套餐封面 + 套餐名

### Sections & Components
1. 顶部封面区
   - 大图封面 + 标题/简述（按现有）
2. 媒体区
   - 图片：轮播 Swiper（多图）+ 点击预览
   - 视频：视频卡片（建议单独区域，避免与图片混播导致体验不一致）
3. 选项选择区
   - Tabs/横向滚动 Chips；选中态高亮
   - 切换时刷新下方“选项资源区”
4. 选项资源区
   - 图片网格（2 列）+ 预览
   - 文件列表（单列）
   - 空态占位：图标 + 文案 + 返回/切换建议
5. 状态与容错
   - 加载中 Skeleton
   - 资源加载失败：提示“加载失败，点击重试”
